# History

The History page shows all the events a contract has been involved in.

The following events may occur:
* Draft created
* Contract created
* Updated (Contract updated)
* Contract imported
* Contract closed
* Contract re-activated

The following changes to the contract’s details are registered, together with any version comments specified for the change:
* Added
* Updated
* Removed
